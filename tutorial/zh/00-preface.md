# 前言 —— 从聊天框走出来

_上次校对：2026-05，对齐 Claude Code v2.x_

你用过 Claude.ai 或者 ChatGPT 网页版。你知道往里打一个问题、看着答案冒出来是什么感觉。也许你甚至已经付了费、跟其中一个工具相处得很自然。但你开始撞到一些墙：

- 它能读你粘进去的内容，但没法直接打开你电脑上那个装着草稿的文件夹。
- 它能告诉你该敲什么命令，但没法替你敲、也看不到结果。
- 你可以用同样的方式 prompt 它五十次，但你**没法把这个 pattern 存下来**，让 AI 在合适的时候自动应用它。

这本教程就是讲怎么翻过这几堵墙。

## 这本教程为什么存在

Claude Code 是 Anthropic 出的 CLI agent —— 还是那个 Claude，但跑在你的终端里，不是网页上。它能：

- 直接读写你项目里的文件
- 跑命令、读取输出
- 被"教会" pattern —— 叫作 **skill** —— 在合适的时候自动应用它们

如果你从没打开过终端，上面这句话里至少有三个词需要解释：**CLI**、**终端**、**skill**。读完这本书，你会知道它们都是什么意思。你会写出自己的第一个 skill。你会用一个叫"渐进式展开"（progressive disclosure）的技巧把它从一个文件升级到多个文件。你还会知道怎么让 Claude 帮你设计新的 skill。

## 你读完会做什么

- 在 Mac 上从容地打开终端，不再慌
- 装好 Claude Code，在 CLI 里跟它进行第一次对话
- 建一个 Claude 能读写的项目
- 手写一个 `SKILL.md` 文件，看着 Claude 自动触发它
- 把这个 skill 重构成多文件、采用渐进式展开
- 用 `/skill-creator` 通过对话设计新的 skill

## 前置条件

这本书对自己不覆盖的事情很诚实。

- **一台 Mac。** 本书 macOS only。Windows 和 Linux 读者不在范围内 —— 详情见 [ADR-0001](../../docs/adr/0001-tutorial-scope.md)。
- **能稳定访问 `google.com` 和 `anthropic.com`。** 如果你所在地区上不去，这件事你要在开始第 2 章**之前**自己解决。本书不教 VPN 怎么搞。
- **一张 Visa 或 Mastercard。** Claude Code 通过 Anthropic 收费。本书推荐订 Claude Pro 作为最省事的路径 —— 撰写时大约每月 US$20。

如果上面三条任意一条不满足，请就此打住。这本书没法帮你跨这些坎，提前告诉你比把你卡在第 2 章更负责任。

## 怎么读这本书

每一章都遵循相同的结构：

- 章首一行小标题，写明这一章上一次校对的时间，以及对齐的 Claude Code 版本
- 讲解概念的正文
- 🛠 **试一试** 框 —— 你在自己电脑上做完再往下读
- 一句话引到下一章

**试一试** 框是最重要的部分。光读不动手，你不可能从第 0 章走到能写出 skill —— 你必须真的去敲。每个框都有一行"你应该看到……"，让你能在继续之前验证结果对不对。

如果章首的校对时间已经是几个月以前的了，请对这一章的命令保持一点警惕 —— Claude Code 每一两周就出新版本。大多数东西不会变，但如果你看到对不上的地方，先去查 Anthropic 官方文档，再下"教程坏了"的结论。
