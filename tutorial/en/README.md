## How to read

The "last calibrated" line at the top of each chapter shows when the chapter was last verified on a clean Mac, and against which Claude Code version. If the timestamp is more than a few months old, commands may have shifted slightly; consult Anthropic's official documentation if something doesn't match.

If you want to know how this book is designed before reading it:

- [`CONTEXT.md`](../../CONTEXT.md) — terminology glossary, reader archetype, chapter conventions
